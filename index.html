Observable Playground

Learn RxJS before deploying to production (for great good, of cource)

This tool is aimed at assisting you in understanding observables by playing with them. There are some code examples prebuilt for you in the left menu. Further, you can modify any of them to see how it changes output. Once a snippet is changed -- resulting output would be recalculated IMMEDIATELY.

NOTE: currently there are following limitations:
  • - timeline is cut to 1000ms
  • - Promises are not supported

Please, check the source codes of this project at github.

Your feedback is welcome!


This tool was inspired by rxmarbles.com, learnrxjs.io and great talks by Bret Victor


const { chart } = require('rp-api'); const { timer } = require('rxjs'); const { take, map } = require('rxjs/operators'); const msg = 'awesome'; timer(0, 5) .pipe( take(msg.length), map(index=>msg[index]) ) .subscribe(chart.createRxObserver());
